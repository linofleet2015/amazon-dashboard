⚙️ GitHub連携設定

月次データを全員に共有するために、GitHubのPersonal Access Token（PAT）を設定します。トークンはこのPCのブラウザにのみ保存され、他の人には見えません。

トークンの作成手順：
GitHub → Settings → Developer settings →
Personal access tokens → Fine-grained tokens → Generate new token
・Repository access: amazon-dashboard のみ
・Permissions → Contents: Read and Write
